How fast can you finite element using scripting languages?

Writing a high performance finite element code for nonlinear coupled problems (e.g., electro-magneto-mechanics) using scripting languages is a challenging task due to conflicting requirements (e.g., speed vs readibility). In this blog, I will discuss my experience on the manipulation of finite element vectors/matrices/tensors using scripting languages such as Matlab and Python (in progress).
